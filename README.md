# Kaggle Data Science Survey – Data Cleaning Project

## Objective
Clean and analyze a real-world survey dataset containing missing values, duplicates, and inconsistent responses.

## Dataset
Kaggle Data Science Survey (2017–2021)

## Tasks Performed
- Removed duplicate records
- Handled missing values
- Selected relevant columns for analysis
- Cleaned categorical variables
- Applied label encoding/mapping
- Generated insights from survey responses

## Tools Used
Python  
Pandas  
NumPy  
Google Colab

## Outcome
The cleaned dataset enabled meaningful analysis of respondent demographics, tools used in data science, and career interests.
